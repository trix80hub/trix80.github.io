First, with your document you will need to start with <!DOCTYPE html>.
this will initiate the html document. W3Schools Link

The html tag is needed in the html file.
<html> and </html> W3Schools Link

The header is a title at the start of the page.
use <head> and </head> to use it.
for example, <head>hello</head> would type "hello". W3Schools Link

The title tag is used to set a title for the page in the tab.
The tag is <title></title>, and you would put the page name in the tags.
For example, <title>My page</title> W3Schools Link

The body tag is used to set up the body where you will write your text.
The tag is <body></body>. W3Schools Link

The header tag is used to make a header for the page.
The tag is <h1></h1>, remember that this goes on to h6.
For example, <h1>Heading!</h1>

The tag is <p></p>.

The comments tag is used to make a comment that doesn't show up in a page.

The a tag is used to make a link for the page.
